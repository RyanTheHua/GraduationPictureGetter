Getting Started

Download the .java file, use VS Code or a compiler to run

After running the file, enter the HD zoomed-in graduation picture from a site like flash photography ( I do not know about any other sites working)

Open the zoomed-in HD picture in its own tab and look at the URL. It should look like &O=27341768&R=10001&F=0276&A=71994&rand=0.8262668656003157 
and is right after the value for the y coordinate

It will create a folder called pictures, and the combined.png is the final, complete preview of the grad picture.
